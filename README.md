# ezmesure-admin

## Install

```sh
$ git clone https://github.com/ezpaarse-project/ezmesure-admin.git
$ cd ezmesure-admin
$ npm i -g
```

## Usage

```bash
$ ezmesure-admin --help
```

## Global options

| Name | Type | Description |
| --- | --- | --- |
| -t, --timeout | String | Request timeout in milliseconds |
| --version | Boolean | Print the version number |
| --help | Boolean | Show some help |
